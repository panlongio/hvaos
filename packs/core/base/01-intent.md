# 01 - 意图文档：我们要做什么 (Intent)

```yaml
intent_spec:
  objective:
    project_name: "{{PROJECT_NAME}}"
    mission: "{{PROJECT_MISSION}}"
    target_audience: "{{TARGET_AUDIENCE_1}}"
    core_pain_point: "{{USER_PAIN_POINT}}"
  anti_objective:
    - "{{ANTI_GOAL_1}}"
    - "{{ANTI_GOAL_2}}"
  success_criteria:
    - "{{SUCCESS_METRIC_1}}"
    - "{{SUCCESS_METRIC_2}}"
  priority:
    north_star_metric: "{{SUCCESS_METRIC_1}}"
    secondary_metric: "{{SUCCESS_METRIC_2}}"
```

## 1. 项目定义

- **项目名称**：`{{PROJECT_NAME}}`
- **核心使命 (Why)**：`{{PROJECT_MISSION}}`
- **北极星指标 (Only One)**：`{{SUCCESS_METRIC_1}}`
- **非目标用户**：`{{TARGET_AUDIENCE_2}}`

## 2. 目标用户与核心痛点

- **目标用户**：`{{TARGET_AUDIENCE_1}}`
- **核心痛点**：`{{USER_PAIN_POINT}}`

## 3. 边界定义

- **关键差异点**：
  1. `{{DIFFERENTIATOR_1}}`
  2. `{{DIFFERENTIATOR_2}}`
- **明确不做**：
  - `{{ANTI_GOAL_1}}`
  - `{{ANTI_GOAL_2}}`

## 4. 完成判定

- **成功标准 1**：`{{SUCCESS_METRIC_1}}`
- **成功标准 2**：`{{SUCCESS_METRIC_2}}`
